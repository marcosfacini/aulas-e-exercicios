como instalar modulos python no vscode:!

1: rodar o cmd como administrador

2: escreva: cd + path (sem o final/python)
cd C:\Users\Marcos\AppData\Local\Programs\Python\Python39-32

3: escreva: python -m pip install (nome do arquivo)
python -m pip install emoji

Se precisar atualizar o instalador primeiro escreva:
python -m pip install --upgrade pip

4: escreva cd Scripts

5: escreva pip install + nome do arquivo
pip install emoji
